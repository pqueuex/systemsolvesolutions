Remote Support Guide

Simple, affordable tools to get help on Windows and macOS from System Solve Solutions.

Start by contacting us at 352-350-5553 or info@systemsolvesolutions.com. We’ll schedule a session and provide the access code needed to connect securely.

Before we connect, close sensitive information and make sure you know the administrator password for your computer. A stable internet connection is essential for a smooth session.

Quick Assist Help macOS Screen Sharing Chrome Remote Desktop

Windows Support

Preferred: Microsoft Quick Assist (Built In)

  1. Press the Windows key and type Quick Assist, then open the app.
  2. Select Get help. When prompted, enter the 6-digit security code we provide and choose Submit.
  3. Accept the permission prompts so we can view your screen. When you’re ready, choose Allow to grant control.
  4. Stay at your device during the session so you can answer questions or approve any additional prompts.

Alternate: Chrome Remote Desktop

  1. Open Google Chrome (or Microsoft Edge) and visit remotedesktop.google.com/support.
  2. Under Get Support, choose Download to install the helper extension, then run the downloaded installer.
  3. Once installed, click Generate Code and share the code with us by phone or email. Codes expire after 5 minutes for security.
  4. Leave the browser window open until we connect. You can end the session at any time.

macOS Support

Preferred: Messages Screen Sharing (Built In)

  1. Open the Messages app and start a conversation with the Apple ID we provide.
  2. Select the Details button, then choose the Screen Sharing icon.
  3. Click Invite to Share My Screen. We’ll accept the request and prompt you when we’re ready to assist.
  4. When prompted, approve remote control so we can help directly, or keep it view-only if you prefer to follow along.

Alternate: Chrome Remote Desktop

  1. Install Google Chrome if it isn’t already on your Mac, then go to remotedesktop.google.com/support.
  2. Click Download to add the Chrome Remote Desktop app and follow the on-screen installation steps.
  3. Choose Generate Code and share the one-time code with us. Grant screen recording and accessibility permissions when macOS prompts you.
  4. Keep the Chrome window active during the session. You can revoke access instantly by closing the connection.
Need a hand setting up? If any step is unclear, stop and give us a quick call. We can stay on the phone while you install the tool or read the code to us to keep things simple. Remote support sessions are billed in 15-minute increments with no additional software fees.

Troubleshooting Checklist

  • Verify your device is connected to the internet and not on a guest Wi-Fi that blocks screen sharing.
  • Disable any VPN temporarily if the remote tool can’t connect.
  • On macOS, grant screen recording and accessibility permissions in System Settings > Privacy & Security.
  • Keep your phone nearby so we can coordinate if the session disconnects.
  • When the work is complete, close the remote tool to end the session and restore full control to you.
